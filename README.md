# Zubayr Production — Portfolio

Mobilograf, videograf va reels editor **Zubayr** uchun shaxsiy portfolio sayt.

- **Uslub:** Apple-style 2026 minimalizm, monoxrom (qora / oq / kulrang), dark theme
- **Stek:** Vanilla HTML + CSS + JS — build tool kerak emas, istalgan static hostingda ishlaydi
- **Tillar:** UZ / RU / EN (navbar'dagi almashtirgich, tanlov `localStorage`da saqlanadi)
- **Shrift:** Inter Tight + Inter (Apple qurilmalarda SF Pro fallback)

## Lokal ishga tushirish

```bash
python -m http.server 8734
# http://localhost:8734
```

## Tuzilma

```
index.html      — yagona sahifa (barcha seksiyalar)
css/style.css   — uslublar (responsive + reduced-motion)
js/i18n.js      — UZ/RU/EN tarjimalar va til almashtirgich
js/main.js      — scroll reveal + nav holati
```
